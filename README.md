## 개요
* 스팀의 한국 게임 목록 입니다.



## 작성 대상
* 플레이 가능한 버전이 "대중" 에게 공개된적이 있는 한국산 게임.
* 자회사 또는 해외 스튜디오가 개발해도 한국산 게임으로 인정.



## 필요 정보
* 한글 제목, 영어 제목, 게임 링크, 출시년도, 장르, 개발사, 퍼블리셔



## 작성 방법
* 저장소 내의 "game_list.ods"( libreoffice spreadsheet ) 파일을 연다.
* "list" sheet에 데이터를 기입 한다.
* Macro 를 작동시켜 "README.md" 파일을 갱신한다.



## 제보
* 상단에 링크되어 있는 Discord 에 남겨 주세요.
